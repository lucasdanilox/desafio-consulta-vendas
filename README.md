# Sobre o projeto
Trata-se de um projeto de vendas (Sale) e vendedores (Seller). Cada venda está para um vendedor, e um vendedor pode ter várias vendas.

# Tecnologias utilizadas
## Back end

* Java
* Maven
* Spring Boot
* Spring Data JPA
* Hibernate
* H2 Database
* Git e Github
